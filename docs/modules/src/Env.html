Env¶

Usage

use Env;

The Env module provides procedures and an iterator which can help to manage environment variables.

proc locale.setEnv(varName: string, varValue: string): int¶

Set an environment variable to value.

Arguments:
  • varName : string – The name of the environment variable to set.
  • varValue : string – The new value of the environment variable.
Returns:

Returns 0 on success, -1 on error.
Return type:

int
proc locale.unsetEnv(varName: string): int¶

Delete an environment variable.

Arguments:varName : string – The name of the environment variable to delete.
Returns:Returns 0 on success, -1 on error.
Return type:int
proc locale.getEnv(varName: string, defaultValue = ""): string¶

Get the value of the environment variable, using default if not set.

Arguments:
  • varName : string – The name of the environment variable to retrieve.
  • defaultValue : string – The value to use if the variable does not exist.
Returns:

The value of the variable
Return type:

string
iter locale.envs()¶

Iterator to iterate over all defined environment variables

Yields:A string containing a record of environment variable
var envs: stringmapstring¶

A map that stores state of envs

Note

The map is populated when the module is imported and does not reflect the realtime state. Also, this is not locale aware. Use methods if you need to run on multiple locales.
